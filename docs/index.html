📘 Como usar o bot

🎲 Comando de dados (roll)

Use para rolar dados em RPG ou cálculos.

  • 1d20 → 1 dado de 20 lados
  • 2d6+3 → 2 dados + 3
  • 1d10! → dado com explosão
  • 3#1d20+5 → repete 3 vezes

🧮 Comando matemático (math)

Resolve expressões matemáticas.

  • 2+3*4 → 14
  • (2+3)*4 → 20
  • 2^4 → potência
  • 16z2 → raiz quadrada
  • 25%200 → porcentagem

⚡ Dicas

  • Use !roll ou aliases configurados
  • Use !math para contas
  • Você pode combinar dados + matemática
